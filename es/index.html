ReanEmu

Software privado de Emulación para servidor de WoW en la versión 3.3.5a. Fork de TrinityCore, pero con varios arreglos de otros repos y especialmente del equipo de desarrollo internacional envuelto en el proyecto ReanEmu. Las raids de ICC, Ulduar y Santuario Rubí están incluidas, también hay muchas misiones y hechizos arreglados, así como algunas características interesantes como el script del espectador de arenas o el sistema de raid-info.
