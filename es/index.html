ReanEmu

Software de Emulación para servidor de WoW en la versión 3.3.5a. Fork privado de TrinityCore con varios arreglos de otros repos y especialmente del equipo de desarrollo internacional envuelto en el proyecto ReanEmu. Las raids de ICC, Ulduar y Santuario Rubí, muchas misiones y hechizos arreglados, así como algunas interesantes características extra, como el arena-espectador y el sistema de raid info, están incluídas.
