The ReanEmu project

ReanEmu is an MMORPG framework used to emulate a World of Warcraft server in the 3.3.5a version, it is built using C++, and using a back-end of MySQl for the DB content. ReanEmu is a private fork of TrinityCore and has been created by joining various famous repos and our own code with the purpose of achieving a complete emulator keeping in mind our main goals: usability, stability and playability. We have designed our own methodology and workflow, based on git-flow, that provides us the capability to keep our code up to date with the updates and improvements of TrinityCore as well as an effective way to handle our release process and day-to-day development.
