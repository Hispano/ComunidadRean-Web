ReanEmu

This is a private Emulation Software for WoW Server Version 3.3.5a. It is a fork of TrinityCore, but has several fixes from other repos and specially from the international Developer-Team involved in the ReanEmu project. ICC, Ulduar and Ruby Sanctum raids are included, there are also lots of fixed quests and spells, as well as some nice features like the arena-spectator script or the raid info system.
